== Description ==

"Restaurant Table Booking Manager" plugin is to manage your online Table booking services for your site and your clients will be able to check availability and book online.
you can show and manage your tables/table seats availability for rent.
You can manage the bookings (availability) on hourly basis. tables and booking calendar page will be created automatically and shortcode to generate
calendar will be automatically copied. you can add/edit tables or delete bookings from admin to manage the full system.

Your Customers will be able to:

= FEATURES: =
- Make bookings in friendly booking interface - select the date(s) and fill form fields.
- Prevent of double booking for already reserved timeslot (1 booking per timeslot for a particular table).

= KEY BENEFITS: =
- very easy to install and configure. 
- Very flexible functionality. Fit to very wide range of business.
- All bookings and settings are stored in your DB. You don't need third party account(s).
- An easy to use Booking Admin Panel that displays bookings in Calendar Overview and lets you manages bookings.
- Built with jQuery, Ajax and other technologies.
- Easy to install and integrate into your site. because necessary post or page will be automatically created and shortcodes will be automatically copied to page.

= Manage your Table Bookings in Admin Panel: =
- Comfortable Admin Panel for booking management. View table bookings in Calendar Overview Panel (Timeline) with possibility to set Day/Week/Month view 
	or in Booking Listing Table with pagination.
- In manage booking Search the booking(s) by different parameters, using the Filter in Admin Panel.
- Pagination of the booking listing.
- Administrator can edit or Delete specific bookings.
- View the bookings in booking calendar of any month of any year.

== Installation ==

1) Copy/Upload 'gen-restbl-uststablebooking' folder to the '/wp-content/plugins/' directory
2) Activate 'Restaurant Table Booking Manager' from wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) create tables for booking.
